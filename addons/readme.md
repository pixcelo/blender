# アドオンの作り方

## 設定
編集→プリファレンス→開発者オプションにチェックすると、
実験的タブが表示され、デバッグオプションが選択可能になる

## WatchAddon
アドオンを変更・保存するだけで、開発中のアドオンの機能を確認できる
